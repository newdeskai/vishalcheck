A: 'Docket: React Notes & Wiki' is a powerful documentation tool built with React, designed exclusively for Odoo. It offers advanced note-taking and wiki functionalities, enhancing the user experience with a modern React UI. The module seamlessly integrates with Odoo, requiring no external server.

Q: Which versions of Odoo does the app support?

A: Our app is compatible with Odoo versions 12, 13, 14, 15, and 16. We've designed it to work smoothly with both Odoo Enterprise and Community editions.

A: Yes, 'Docket: React Notes & Wiki' is versatile and can be used on both Odoo SH and private servers, ensuring flexibility and convenience for all users.
